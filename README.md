# AlgoCrack

## Introduction
AlgoCrack is a cheating website that imitates leetcode.

## Feature
- Google Oauth login
- Upload questions
- Use code runner to execute code
- Share solutions with others
- Get answers from ChatGPT
- Get past submission information

## run Project
```bash
# use node v20
$ cd backend
$ yarn install
$ yarn run build
$ yarn run start
```

## Debug Mode
